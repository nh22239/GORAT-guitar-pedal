# GORAT RAT clone 
Here at **GORAM** industries, we love the sound of a good dirty **RAT**, but what if we all want to use one at the same time? Suffice to say, we'd probably all get lost in eachothers sound.
That's why we've decided to go custom, a brand new **RAT** clone built by **GORAM**, for **GORAM**.

## New features:
### Frequency response selector switch!

  A simple knob that allows switching the filters of the op-amp stage between **Regular**, **Bass**, and **BEANS** modes
  
### Diode selector switch

  Pick between 6 different diode combinations
  1. BAT41 - BAT 41
  2. Red LED - Red LED 
  3. 1N914 - 1N914
  4. 1N914 - Open
  5. BAT41 - 1N914
  6. BAT41 - Red LED

### Wet/Dry knob
  Does what it says on the tin, for when the un-adulterated bass tone needs to shine through a bit more.

  
## Future Features:
### Diode selector gain compensation
  When switching diodes, also switch feedback resistor on a compensation op-amp.
### SMD components
  The enclosure is very busy, any new features would probably require this.
### Mini versions
  Much smaller package sizes individualised for Bass, Standard, and Beans.
